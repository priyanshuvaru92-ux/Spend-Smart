Build a complete React web app named **"SpendSmart"** — an AI-powered expense tracker for Indian college students.

Use:

* Vite + React
* Tailwind CSS
* Recharts
* Lucide React
* localStorage
* Google Gemini 2.0 Flash API

---

## 🎯 CORE FEATURES

* Add expenses (name, amount ₹, category, date)
* Dashboard with totals + charts
* Expense list with delete
* AI analysis using Gemini
* Intro screen (first visit only)
* How It Works page

---

## 💾 DATA (IMPORTANT)

Use localStorage key: "spendsmart_expenses"

Expense object:
{
id: crypto.randomUUID(),
name,
amount: Number,
category,
date ("YYYY-MM-DD")
}

Create a hook useExpenses that provides:

* expenses

## 🧾 ADD EXPENSE (CRITICAL FIX SECTION)

Use a modal with controlled state:

const [form, setForm] = useState({
name: "",
amount: "",
category: "",
date: new Date().toISOString().split("T")[0]
});

### ✅ CATEGORY (IMPORTANT)

* Use controlled component
* Either:

  * Proper `<select>` with value + onChange
    OR
  * Custom dropdown (must update state correctly)

onChange:
e => setForm(f => ({ ...f, category: e.target.value }))

### ✅ DATE (IMPORTANT)

<input
type="date"
value={form.date}
max={new Date().toISOString().split("T")[0]}
onChange={e => setForm(f => ({ ...f, date: e.target.value }))}
/>

### ✅ VALIDATION

Check:

* name not empty
* amount > 0
* category selected
* date exists

### ✅ SUBMIT

* Add expense
* Reset form
* Close modal
* Show toast

## 🤖 AI ANALYSIS (GEMINI)

Use API key:
const API_KEY = import.meta.env.VITE_GEMINI_API_KEY

If missing:
Show warning banner (don’t crash)

### API CALL

POST:
https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent?key=${API_KEY}

Parse:
data.candidates[0].content.parts[0].text
