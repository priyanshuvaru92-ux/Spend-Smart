Build a complete React web app named **"SpendSmart"** — an AI-powered expense tracker for Indian college students.

Use:

* Vite + React
* Tailwind CSS
* Recharts (charts)
* Lucide React (icons)
* localStorage (data)
* Gemini 2.5 Flash API (for AI)

---

## 🎯 CORE FEATURES

1. Add expenses (name, amount ₹, category, date)
2. View dashboard with:

   * Total spent
   * Top category
   * Transaction count
3. Charts:

   * Donut chart (category breakdown)
   * Bar chart (last 7 days)
4. Expense list:

   * Delete with inline confirm
   * Search + category filter
5. AI Analysis (Gemini API):

   * Insights
   * Savings tips
   * Warnings
6. Intro screen (first visit only)
7. "How It Works" section

---

## 🤖 AI ANALYSIS (IMPORTANT)

Use Gemini API:

POST:
https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent?key=YOUR_API_KEY

Headers:
Content-Type: application/json

Body:
{
"contents": [{
"parts": [{
"text": "Analyze this expense data and give insights, tips, and warnings:\n\nTotal: ₹X\nCategories: ...\nRecent: ..."
}]
}]
}

Rules:

* Only enable if ≥ 3 expenses
* Show loading state
* Parse response into 3 sections:

  * Insights
  * Tips
  * Warnings

If API key missing:
Show warning banner

Use:
import.meta.env.VITE_GEMINI_API_KEY
